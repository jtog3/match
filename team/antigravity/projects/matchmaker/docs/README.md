# LoL Team Matchmaker - Premium Edition

## プロジェクトの目的
League of Legends (LoL) のカスタムマッチなどで、プレイヤー間の実力差を考慮し、最適なチーム分けを自動で行うためのウェブアプリケーションです。

## 主な改善点
- **直感的な操作**: 従来のドロップダウン選択式から、スライダー形式で「強さ」を視覚的に調整できるように変更。
- **プレミアムデザイン**: SVGアイコン、グラデーション、グラスモルフィズム、マイクロアニメーションを活用した、ゲーム体験を向上させるモダンなUI。
- **マッチングロジック**: プレイヤーの希望レーンと強さを考慮し、最適な勝率バランス（30%〜70%）を維持しつつ、チーム分けを行います。

## フォルダ構成
- `antigravity/projects/matchmaker/docs`: プロジェクト管理・設計ドキュメント
- `antigravity/projects/matchmaker/code`: 実行可能な全ソースコード

## セットアップ
ブラウザで `index.html` を開くだけで動作します。外部ライブラリはCDN経由で取得するか、標準機能のみで実装しています。
